# Versión breve de preguntas esenciales

## Datos básicos
- ¿Cómo se llama?
- ¿Cuál es su fecha de nacimiento?
- ¿Tiene tarjeta sanitaria o documento identificativo?
- ¿Cuál es su teléfono de contacto?
- ¿Quién le acompaña?

## Motivo de consulta
- ¿Qué le ocurre?
- ¿Desde cuándo?
- ¿Cómo empezó?
- ¿Ha empeorado?
- ¿Tiene dolor? ¿Dónde? ¿Cuánto de 0 a 10?

## Antecedentes
- ¿Qué enfermedades tiene?
- ¿Le han operado alguna vez?
- ¿Tiene alergias?
- ¿Toma medicación habitual?

## Situación actual
- ¿Ha tenido fiebre?
- ¿Tiene dificultad para respirar?
- ¿Tiene náuseas, vómitos o diarrea?
- ¿Ha sangrado?
- ¿Ha perdido el conocimiento?

## Contacto y apoyo
- ¿Con quién vive?
- ¿Tiene una persona de contacto?
- ¿A quién avisamos si hace falta?

# Nota

Este formulario es una plantilla orientativa y no sustituye los documentos oficiales de cada hospital o servicio de salud.
